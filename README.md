# Vickers Test Ontology (VTO)

Vickers Test Ontology (VTO) has developed for representing the Vickers testing process, testing equipment requirements, test pieces charactristics, and related testing parameters and their measurement procedure according toÂ DIN EN ISO 6507-1 standard.

More information can be found at http://obofoundry.org/ontology/vto
